Blog Project

A simple and scalable blogging platform built for writers and readers.

Project Overview

The Blog Project is a platform that enables users to create, read, update, and delete blog posts. It offers a simple, user-friendly interface and provides functionality for managing posts, adding categories, and searching content. Ideal for personal blogs or small-scale publications.

Key Features

  • Blog Post Creation and Editing
  • Comments Section
  • Category-based Post Organization
  • Post Search Functionality
  • Responsive and Mobile-Friendly Design
  • User Authentication (Login/Sign-up)
  • Admin Panel for Content Moderation

Installation

To set up this project locally on your machine, follow these steps: 

      
        1. Clone the repository:
        git clone https://github.com/lokeshgoud15/Mern-Blog-App.git
  
        2. Navigate into the project directory:
        cd Mern-Blog-App
  
        3. Install dependencies:
        npm install
  
        4. Start the development server:
        npm start

Technologies Used

This project is built using the following technologies:

  • Tailwind CSS
  • React.js
  • Node.js
  • Express.js
  • MongoDB
