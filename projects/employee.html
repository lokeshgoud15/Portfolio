Employee Project Management

A platform for managing employee projects and tasks effectively.

Project Overview

This project aims to create an application that helps organizations manage their employee's projects, track progress, and improve productivity. It provides functionalities for task assignment, project tracking, and team collaboration.

Key Features

  • Task Assignment
  • Project Tracking
  • Real-time Team Collaboration
  • User Authentication
  • Progress Reports and Dashboards
  • Customizable Notifications

Installation

To set up this project locally on your machine, follow these steps: 

      
        1. Clone the repository:
        git clone https://github.com/lokeshgoud15/Employee-Management-System.git
  
        2. Navigate into the project directory:
        cd Employee-Management-System
  
        3. Install dependencies:
        npm install
  
        4. Start the development server:
        npm start

Technologies Used

This project is built using the following technologies:

  • Tailwind CSS
  • React.js
  • Node.js
  • Express.js
  • MongoDB
